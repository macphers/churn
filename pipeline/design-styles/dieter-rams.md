# Dieter Rams

> "Good design is as little design as possible."

## Principles

- **Less, but better.** Strip every element that doesn't serve function.
  If removing it changes nothing, it shouldn't exist. White space is a
  feature, not a waste.
- **Honest.** Don't make a button look like something it isn't. Don't
  animate to distract. Don't use decoration to disguise emptiness.
  The interface should communicate exactly what it does.
- **Useful.** Every pixel earns its place by helping the user complete
  a task. Ornament that doesn't inform is noise.
- **Unobtrusive.** The design serves the content, not itself. The user
  should remember what they accomplished, not what the interface
  looked like.
- **Thorough down to the last detail.** Alignment, spacing, and
  typography are not afterthoughts. Inconsistency at 1px is still
  inconsistency.
- **Long-lasting.** Avoid trends. A good interface should look
  appropriate in 5 years, not just this quarter.

## Design Review Focus

When reviewing design in this style, prioritize:
1. Is every visual element functionally justified?
2. Is the typography hierarchy clear and minimal?
3. Does spacing follow a strict, consistent scale?
4. Are there any decorative elements that don't communicate information?
5. Would this design look dated in 2 years?
