# Brutalist

> "The raw material of the web is text and links. Everything else is decoration."

## Principles

- **Raw structure visible.** Don't hide the bones. Borders are hard
  lines, not soft shadows. Backgrounds are flat colors or nothing.
  The grid is visible and unapologetic.
- **System fonts only.** No Google Fonts, no custom typefaces. Use
  the system stack — it loads instantly and looks honest.
- **High contrast.** Black text on white. White text on black. Color
  is accent, not wallpaper. When you use color, make it scream.
- **No gratuitous animation.** Motion is for state change, not delight.
  Hover states change instantly. Transitions are 0ms or 100ms max.
- **Dense information.** Pack more in. Respect the user's time by
  showing more data per viewport. Scrolling is acceptable; hunting
  through pages is not.
- **Ugly is fine.** Functional ugly beats decorative pretty. If the
  design serves the task, it's good design — regardless of whether
  it photographs well.

## Design Review Focus

When reviewing design in this style, prioritize:
1. Is the layout honest about its structure?
2. Are there unnecessary visual embellishments (shadows, gradients, rounded corners)?
3. Is information density maximized?
4. Does animation serve a functional purpose or is it decorative?
5. Would this work with CSS disabled?
